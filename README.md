# Система для управления рецептами

Позволяет создавать ингредиенты, рецепты, увеличивать порции и генерировать список покупок.

## Использование

Перед запуском убедитесь, что у вас установлен Python 3.7 или выше.

1. Откройте терминал (командную строку)
2. Перейдите в папку с проектом с помощью команды cd (например: cd C:\путь_к_папке)
3. Выполните команду: python DietaryRecipe.py

После запуска вы увидите результат работы программы - примеры создания ингредиентов, рецептов и списка покупок.
