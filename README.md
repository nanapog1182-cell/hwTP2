# Система для управления рецептами. 
 
Позволяет создавать ингредиенты, рецепты, увеличивать порции и генерировать список покупок. 
 
## Использование 
 
Запуск кода: 
 
Откройте терминал, перейдите в папку D:\TPhw и выполните команду: python DietaryRecipe.py 
 
Требуется Python 3.7+
